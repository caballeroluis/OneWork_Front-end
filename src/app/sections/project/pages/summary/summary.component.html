About the Project

Onework is a project made and deployed by {{ names[0] }} and {{ names[1] }}, in order to add precision and agility to their hiring process work and that of their friends. Nonprofit.

It is made with Node Express, Mongodb and Angular. Deployed in nginx on Linux.

Version control has been done with git and project management with gitlab.

The server is programmed to save energy by automatically turning itself off at night. Sorry for the inconvenience.

Components used in the assembly of this server:

Processor: AMD Ryzen 5 1600 Stepping AF 3.6GHz BOX

Storage: WD Green 3D SSD 120GB SATA3

Memory (x2): Corsair Value Select DDR4 2133 PC4-17000 8GB CL15

Mother board: MSI A320M-A PRO

Graphics: Zotac GeForce GT710 1GB GDDR3

Case: Tacens Anima AC016 USB 3.0

Extra fan: Tacens Aura II 80x80

PSU: Corsair VS450 450W 80 Plus

Tools: Crimper

Tools: UTP CAT6 cable

Tools: UTP Cat.6 RJ45 conector
