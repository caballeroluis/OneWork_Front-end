About the Project

Onework is a project made and deployed by {{ names[0] }} and {{ names[1] }}, in order to add precision and agility to their hiring process work and that of their friends. Nonprofit.

It is made with Node Express, Mongodb and Angular. Deployed in nginx on Linux.

Version control has been done with git and project management with gitlab.

The app is in the first sprint of development. Soon we are going to improve UX UI, depuration, language translations and websockets service for live drag and drop in the board and more ambitiously our own videocall service with standard WebRTC.

Thanks to the dev tools, which are activated from the right menu, you will be able to see the local status of the front-end application printed on the console.

The server is programmed to save energy by automatically turning itself off at night.

Thanks for the interest.
