How to use

  1. Sign up

    To create an offer, you must create an account of role Recruiter and login with it. Next, you need to navigate to the user list and select a Worker user to create an offer.

    To be available as a Worker, you will need to create a Worker role account.

  2. Validated account

    Validated users have a blue badge.

    To validate your username, contact the administrators by the following email address: {{ ownerMail }}.

  3. Delete account

    You can delete your account by clicking on the corresponding button in the user list after logging in with your account.
